Introducing Tableau 8.2

New features to help you discover and communicate insights

Tableau Desktop now runs natively on the Mac.

It’s possesses the full analytics package, built to support high-resolution Retina displays, contains Mac-specific controls, and will also work with all of your existing Tableau workbooks.

Create compelling, interactive, data-driven stories.

Assemble sheets and dashboards into a narrative arc that tells the story in your data. Capture key insights or events with annotations, highlights and filters. Add descriptions to emphasize findings or add business specific notations. Make your story interactive to encourage further exploration.

New map designs

Maps are critical to geographic analysis and have received a major overhaul in Tableau 8.2. New map designs produced in collaboration with Stamen mean better views, worldwide detailed levels of zoom, an improved mapping server and support for high DPI displays.

Visual Data Window

Tableau have completely redesigned the data connection experience from the ground up. Connect to multiple tables, add in joins with one click and preview your data to make sure you’ve got what you need. You can now also modify field properties, add data source filters, and extract your data.

Additional Tableau Desktop features

New Excel And Text File Connector

Connect to Microsoft Excel and text file data sources that are more than 255 columns. The new connectors automatically and more accurately detect data types and support functions, including Count Distinct and Median.

Improved SAP HANA Connector

The SAP HANA connector now supports HANA variables and Input Parameters.

Native Support For Google Bigquery API

The Tableau connection to Google BigQuery now uses the native API from BigQuery, which means you’ll see improved performance and flexibility.

Splunk Connector

Connect to and analyze machine-data in Splunk with the native Splunk connector (originally introduced in Tableau 8.1.4).

Additional Tableau Server features

Server Rest API

Tableau Server now comes with a REST API to help you easily manage and change your server resources programmatically via HTTP. You can also use the API to create new sites, add or delete users from sites, and much more.

Simplified Log Access

Tableau 8.2 allows you to download Tableau Server log files directly from the Admin page in your web browser. Key log files are now generated as JSON to be machine-readable.

Import & Export Sites

All content within Tableau Server can be exported and imported on a per-site basis. This allows you to back up more efficiently or move content from one Tableau Server instance to another.

Disable Web Authoring

On Tableau Server, all web authoring capabilities can be disabled on site by site basis, providing more administrative flexibility and control.

Change Content Owner

You can now reassign the ownership of any published content to another user on Tableau Server or Tableau Online.

Responsive Marks

Tableau 8.2 has much faster response times with the web experience of published content performing quicker within the browser. When hovering over a mark on a view, the mark will respond immediately.

OAuth Support

Tableau now supports OAuth authentication when connecting to Salesforce, Google Analytics and Google BigQuery. This makes it far more convenient to connect to these data sources across all your workbooks and data sources. It also adds that all important layer of security as Tableau does not have to hold your credentials directly.
