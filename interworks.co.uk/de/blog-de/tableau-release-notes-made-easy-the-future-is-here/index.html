Tableau Release Notes Ganz Einfach – Die Zukunft Ist Hier

Posted by Fabian Zimmer on 8 May, 2015   This post is also available in:  

This post is also available in: Englisch

Diesen Blog Post möchte ich mit einer einfachen Vermutung beginnen: Wenn Sie Tableau bereits einige Zeit benutzt haben, werden sie bestimmt wenigstens ein Mal durch die Tableau Release Notes auf Tableau’s Webseite gescrollt haben. Dabei haben Sie entweder nach Problemen oder Fehlern, die Sie möglicherweise erlebt haben, gesucht, oder aber einfach nach coolen und neuen Features von neueren Versionen geschaut, die Sie unbedingt ausprobieren wollen.

Bisher musste man pro Version die jeweilige Seite einzeln durchforsten, was nicht besonders viel Spaß macht. Was das auch unmöglich gemacht hat, ist auf der Suche nach einem Stichwort alle Release Notes von verschiedenen Versionen gleichzeitig zu durchsuchen. Nicht verzagen, Hilfe ist unterwegs.

Diese Visualisierung vereint alle Release Notes der letzten, wichtigen Updates und zeigt sie auf schöne Art und Weise auf einer einzigen Seite an. Benutzen Sie den Wildcard Filter um alle gleichzeitig zu durchsuchen, teilen Sie die Release Notes in Kategorien ein und sehen Sie, wie viele neue Features und Fehlerbehebungen im Verlaufe der Zeit implementiert worden sind.

 

Klicken Sie aufs Bild um die Visualisierung auf Tableau Public zu öffnen.

Finden Sie das hilfreich? Lassen Sie es uns in den Kommentaren wissen und schauen Sie öfter für neue Blog Posts vorbei, in denen wir uns mit allen möglichen spannenden Themen rund um Tableau auseinandersetzen. Wenn Sie Fragen haben, schreiben Sie mir einfach eine E-Mail oder, falls Sie Tableau ausprobieren wollen, klicken Sie hier um eine kostenlose 14-tägige Testversion herunterzuladen!
