]]> I have upgraded the ‘ideal’ method for creating a 3 way venn to be more performant and mathematically improved. Check it out here.

I have left this original blog post in as it demonstrates how to use sets in calculations and give you the choice.

The 3 way Venn is an upgrade from my previous post on the 2 way venn. This chart type shows exclusive members of a group (or cohort) and also the overlap where members are shared between groups. The example below, built from Tableau’s demo data, shows customers who have received goods packaged in one or more types of container. This type of ‘cohort based’ visualisation uses sets and compares the members of those sets. As a result the visualisation can take a while to compute. (yes, I’m English and that is how I spell visualisation). Tableau is being very clever so please be patient!

Learn About Tableau

Steps to Build:

    1. Create numeric calculated fields (1 or null) for each container type (MB,JD,JB).
    2. Create a set for each exclusive cohort based on the calculated fields from the previous step. Although Tableau can create combined sets, we are going to do this by using set based calculated fields instead.
    3. Create a set based calculated field for each of the 9 variations. The output should be a dimension e.g. ‘JB’ or ‘JBJD’
    4. This visualisation is a dual axis with all the combined cohorts being in the background so we are going to create 2 layers. Layer 2 contains your 3 exclusive sets (MB,JD,JB) whilst layer 1 contains all the JBJD’s etc.
    5. The placement is based on cartesian coordinates and is static (it is my intention to create dynamic text placement at some stage).
    6. Y position is simply 0, 0.5 or 1 depending on the cohort.
    7. X position Layer 1 is 0.25, 0.5 or 0.75 again depending on the cohort
    8. X position Layer 2 is 0, 0.5 or 1
    9. Now we have built all the calculations simply add the fields to the worksheet not forgetting to synchronise your dual axis.
    10. Make the mark type ‘circles’, add some transparency and a border to layer 2, make all the circles white on layer 1. Set the size to max (depending on your data)
    11. On your dashboard be sure to size the worksheet appropriately
    12. You’re done!

Don’t forget you can download this workbook to see my calculations using the ‘Download’ button at the bottom of the visualisation.
